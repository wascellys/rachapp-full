================================================================================
ENDPOINTS PRINCIPAIS
================================================================================

Base URL: http://localhost:8000/api/v1/

Autenticação:
- POST /api/auth/token/                    (Obter token JWT)
- POST /api/auth/token/refresh/            (Renovar token)

Usuários:
- GET /usuarios/                           (Listar usuários)
- GET /usuarios/me/                        (Dados do usuário autenticado)
- PATCH /usuarios/update_profile/          (Atualizar perfil)

Rachas:
- POST /rachas/                            (Criar racha)
- GET /rachas/                             (Listar rachas)
- GET /rachas/meus_rachas/                 (Meus rachas)
- GET /rachas/{id}/ranking/                (Ranking do racha)

Partidas:
- POST /partidas/                          (Criar partida)
- POST /partidas/{id}/registrar_gol/       (Registrar gol)
- POST /partidas/{id}/registrar_presenca/  (Registrar presença)
- POST /partidas/{id}/finalizar/           (Finalizar partida)

Prêmios:
- POST /premios/                           (Criar prêmio)
- GET /premios/                            (Listar prêmios)

Solicitações:
- GET /solicitacoes/                       (Listar solicitações)
- POST /solicitacoes/{id}/aprovar/         (Aprovar solicitação)
- POST /solicitacoes/{id}/negar/           (Negar solicitação)
